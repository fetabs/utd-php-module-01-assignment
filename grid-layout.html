Hyper Text Markup Language ou “Linguagem de Marcação de Hipertexto” em português, é utilizado para criar toda a estrutura da página através do uso de tags para sinalizar onde cada elemento será implementado.

Cascading Style Sheet, ou simplesmente CSS, é uma linguagem de estilos responsável por definir as características estéticas do site.

Javascript adiciona funcionalidades dinâmicas nas páginas, como movimentos, eventos, além de permitir o processamento e transformação de dados enviados e recebidos. Ele permite criar conteúdos atualizados dinamicamente, em composição às estruturas com HTML de forma estática.
